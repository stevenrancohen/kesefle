מחולל חתימת אימייל

ממלאים פרטים, רואים בפועל איך זה ייראה, מעתיקים בלחיצה — ומדביקים ב-Gmail או Outlook.

פרטים

מילוי טופס

איך זה עובד

  1. תמלאו את הפרטים מעל
  2. תראו את החתימה משתנה בזמן אמת
  3. לחצו על "העתק חתימה"
  4. הדביקו ב-Gmail (הגדרות → חתימה) או Outlook

תצוגה מקדימה

חי

כך זה ייראה

ב-Gmail: הגדרות (גלגל שיניים) → "ראה את כל ההגדרות" → גלילה ל-"חתימה" → הדבק. ב-Outlook: קובץ → אפשרויות → דואר → חתימות.

קוד HTML גולמי

למה מחולל חתימה?

חתימה טובה זה כרטיס ביקור חינמי

תואם לכל לקוח דואר

החתימה נבנתה כטבלת HTML — עובד ב-Gmail, Outlook, Apple Mail, Yahoo.

ללא שמירת נתונים

הכל קורה בדפדפן שלך. אנחנו לא שולחים את הפרטים לאף שרת.
