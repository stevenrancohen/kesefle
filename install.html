הוסף את כספ'לה
למסך הבית של הטלפון

PWA = Progressive Web App — אפליקציה אמיתית, רצה offline, אבל בלי צורך באישור של אפל/גוגל ובלי לחכות 24 שעות להורדה.

iPhone / iPad — דרך Safari

  1. 1

    פתח Safari

    חשוב: חייב להיות Safari, לא Chrome ב-iPhone. iOS לא מאפשר התקנה דרך דפדפנים אחרים.

    גש לכתובת: kesefle.vercel.app

  2. 2

    לחץ על כפתור השיתוף

    בתחתית המסך — אייקון של ריבוע עם חץ למעלה ↑

  3. 3

    גלול ובחר "הוסף למסך הבית"

    (Add to Home Screen) — בתפריט שנפתח, גלול קצת ותראה את האפשרות.

  4. 4

    לחץ "הוסף"

    אם רוצה — שנה את השם ל"כספ'לה" (זה ברירת המחדל). לחץ הוסף.

  5. 5

    ✓ מוכן!

    האייקון של כספ'לה מופיע במסך הבית. לחץ עליו → האפליקציה נפתחת במצב מלא, בלי שורת הכתובת של הדפדפן.

⚠ הערה: אם מתקין דרך Chrome ב-iPhone, האייקון מוסיף קישור לדפדפן ולא PWA אמיתי. Safari = PWA אמיתי.

Android — דרך Chrome

  1. 1

    פתח Chrome

    גש לכתובת: kesefle.vercel.app

  2. 2

    חכה לבאנר ההתקנה

    לרוב Chrome מציג באנר בתחתית: "Install Kesefle / התקן". לחץ עליו ועבור ישר לשלב 4.

    אם הבאנר לא הופיע, המשך לשלב 3.

  3. 3

    תפריט (3 נקודות אנכיות) → "הוסף למסך הבית" / "התקן אפליקציה"

    בפינה הימנית-עליונה של Chrome.

  4. 4

    לחץ "התקן"

    Android מתקין את האפליקציה — היא מופיעה ברשימת האפליקציות וגם במסך הבית.

  5. 5

    ✓ מוכן!

    כספ'לה רצה כאפליקציה אמיתית. עובדת גם offline (מציגה את הדשבורד מהמטמון).

💡 טיפ: Samsung Internet ו-Firefox Android תומכים גם הם ב-PWA. החוויה הטובה ביותר ב-Chrome.

Mac — דרך Safari או Chrome

  1. 1

    פתח את כספ'לה ב-Safari או Chrome

    kesefle.vercel.app

  2. 2A

    Safari: File → Add to Dock

    macOS Sonoma ומעלה. הסמל מופיע ב-Dock — לחיצה פותחת אותו כחלון אפליקציה.

  3. 2B

    Chrome: בשורת הכתובת ← אייקון התקנה

    בצד הימני של שורת הכתובת. נראה כמו ⊕. לחץ → "Install".

  4. 3

    ✓ אפליקציה ב-Dock

    כספ'לה נפתחת כחלון נפרד, עם תפריט משלה. אפשר להזיז ל-Spaces, להוסיף ל-Stage Manager וכו'.

Windows — דרך Chrome או Edge

  1. 1

    פתח את כספ'לה ב-Chrome או Edge

    kesefle.vercel.app

  2. 2

    לחץ על אייקון ההתקנה בשורת הכתובת

    בצד שמאל של ☆ (כוכב). נראה כמו ⊕. לא רואה אותו? תפריט (3 נקודות) → "התקן Kesefle".

  3. 3

    לחץ "התקן"

    Windows מוסיף את כספ'לה לתפריט התחל וגם ל-taskbar (אם תרצה — תוכל להצמיד).

  4. 4

    ✓ אפליקציה אמיתית

    מופיעה ב-Alt+Tab, ב-Task Manager, וב-Settings → Apps. אפשר להסיר כמו כל אפליקציית Windows.

למה PWA ולא אפליקציה בחנות?

  • →
    אין אישור אפל/גוגל — אנחנו לא צריכים לחכות שבועות לאישור עדכון.
  • →
    עדכונים אוטומטיים — אין צורך להוריד גרסה חדשה. תשתמש ב-PWA → תקבל את הגרסה החדשה ביותר.
  • →
    פחות נפח אחסון — האפליקציה תופסת ~1MB במקום 50-100MB.
  • →
    עובדת offline — הדשבורד נשמר בקאש המקומי, פותח גם בלי חיבור.
  • →
    לא משלמים עמלות — אפל לוקחת 30%. עם PWA, 100% מהתשלום של Pro/Family הולך אלינו ולא ל-Apple.

פתרון תקלות

לא רואה אפשרות "הוסף למסך הבית" / "התקן אפליקציה"

בדוק:
• אתה משתמש בדפדפן הנכון (Safari ב-iPhone, Chrome/Edge במחשב, Chrome ב-Android)
• אתה גולש דרך HTTPS (kesefle.vercel.app, לא http://)
• הדפדפן מעודכן לגרסה האחרונה
• ביקרת בכמה דפים באתר (חלק מהדפדפנים מבקשים זאת לפני ההצעה)

האפליקציה מציגה גרסה ישנה גם אחרי עדכון

סגור את האפליקציה לחלוטין (לא רק minimize) ופתח שוב. אם זה לא עוזר — מחק את האפליקציה ותתקין מחדש דרך הדפדפן. ה-Service Worker יעדכן את הקאש לגרסה החדשה.

אני לא רואה את הדשבורד כשאני offline

בדוק שהשתמשת באפליקציה לפחות פעם אחת כשהיית online — אז ה-Service Worker שומר את הדפים. אם תפתח את האפליקציה לראשונה offline, היא לא תוכל להציג כלום (אין מה לטעון).

אני רוצה להסיר את האפליקציה

iPhone: לחיצה ארוכה על האייקון → "מחק אפליקציה". Android: ארוכה → "הסר" או דרך הגדרות → אפליקציות. Mac: גרור מה-Dock → Bin. Windows: Settings → Apps → Kesefle → Uninstall.

עוד לא יצרת חשבון?

הרשמה ב-Google → גיליון פרטי ב-Drive → התחל לעקוב.
