## ✅ מה כבר נבנה (לפי תחומים)

### 🤖 הבוט עצמו (Apps Script + WhatsApp Cloud API)

- **הבנת עברית** — מילון של 18,725+ מילות מפתח לסיווג הוצאות
- **זיהוי קבלות בתמונה** (📸 OCR) — שולחים תמונה של חשבונית, הבוט קורא אותה עם Claude Haiku ושומר אוטומטית
- **זיהוי הודעות קוליות** — Gemini transcript + ניתוח
- **שיחה חכמה (LLM)** — Gemini מבין הודעות לא מובנות ועונה כמו אדם
- **תשלומים בתשלומים** — "ספה 1000 שקל 5 תשלומים" → נשמר אוטומטית כ-5 חיובים קבועים
- **שיטות תשלום** — זיהוי "ביט/מזומן/אשראי" + ברירת מחדל לפי לקוח
- **ריבוי עסקים** — "עסק 1 320 שיווק" / "עסק 2 500 שיווק" → גיליון אוטומטי לכל עסק
- **פיקוד "מחק"** — מחיקת השורה האחרונה מהגיליון של הלקוח
- **שאלון התאמה אישית** — 4 שאלות בהרשמה (סוג מעקב, הוצאות קבועות, אוטומציה, ילדים)
- **תזרים פרואקטיבי** — הבוט מציע "להגדיר כקבוע" אחרי 3 חיובים זהים
- **אזהרות חריגה** — "הוצאת 40% יותר על אוכל השבוע"
- **חגיגת הוצאה ראשונה** — תגובה מיוחדת ללקוחות חדשים
- **דיגסט שבועי** — סיכום אוטומטי בימי ראשון

### 🛡️ אבטחה (קריטי לרב-לקוחות)

- **בידוד בין לקוחות** — כל לקוח כותב רק לגיליון שלו, אף פעם לא לאחר
- **18 בדיקות אוטומטיות** של הבידוד (`bot/test_isolation.js`)
- **חתימה דיגיטלית מ-Meta** מאומתת על כל webhook
- **שיטת אנטי-לולאה** — זוהה לולאה עם Hermes Agent ותוקנה (3 שכבות הגנה)
- **כפתור חירום** (`KFL_DISABLE_BOT_WRITES`) — עוצר את כל הכתיבות תוך שניות במקרה חירום
- **תקרת מענה** — מקסימום 20 הודעות לטלפון ב-60 שניות (הגנה מפני באגים)
- **רוטציית סודות** — KESEFLE_BOT_SECRET מתחלף תקופתית
- **טוקנים מוצפנים** — refresh tokens של Google מוצפנים ב-AES עם מפתח-לכל-משתמש

## 🛠️ איך בנוי הסטאק

- **Frontend:** HTML סטטי + Tailwind CSS (ללא framework, ללא build step)
- **Backend:** Vercel serverless functions, ESM Node.js
- **State:** Upstash KV (Redis)
- **Sheets:** Google Sheets API לכל לקוח דרך OAuth `drive.file`
- **Bot:** Google Apps Script (הדבקה ידנית — לא Vercel)
- **LLM:** Anthropic Claude Haiku (OCR) + Google Gemini (שיחה)
- **WhatsApp:** Meta Cloud API (v21.0)
- **Email:** Resend
- **Billing:** PayPal + Coinbase Commerce
- **חשבוניות:** חשבונית ירוקה
- **בדיקות:** Node.js (300+ בדיקות אוטומטיות)
