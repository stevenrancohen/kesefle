### #294 — bot: link code logged as an expense (merged 2026-06-08)
Steven's own link code `קוד 870549` was recorded as a ₪870,549 expense instead of linking his WhatsApp — up to 6 of 10 signups were stuck unlinked. Fixed by intercepting `(קוד|code|link) NNNNNN` at the very top of the doPost wrapper, before any router. Bot build `2026-06-08-linkfix`.

### #295 — admin: segment activation by registration health (merged 2026-06-08)
`lib/activation.js` now classifies each signup as linked / pending_link / no_sheet and computes retention on the linked segment only, yielding an honest verdict: PLUMBING (links broken) vs VALUE_PROBLEM (linked users don't return) vs OK. Admin card + digest show "X/Y linked returned".

### #297 — bot: first-expense activation fixes (merged 2026-06-09)
