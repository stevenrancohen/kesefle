⚡ Pas de magasin d'applications · Sans téléchargement

Ajoutez כספ'לה
à l'écran d'accueil de votre téléphone

PWA = Progressive Web App — une vraie application, fonctionnant hors ligne, mais sans avoir besoin de l'approbation d'Apple/Google et sans attendre 24 heures pour la télécharger.

iPhone / iPad — via Safari

  1. 1

    Ouvrez Safari

    Important : ce doit être Safari, pas Chrome sur l'iPhone. iOS n'autorise pas l'installation via d'autres navigateurs.

    Rendez-vous à l'adresse : kesefle.vercel.app

  2. 2

    Touchez le bouton de partage

    En bas de l'écran — l'icône d'un carré avec une flèche vers le haut ↑

  3. 3

    Faites défiler et sélectionnez « Sur l'écran d'accueil »

    (Add to Home Screen) — dans le menu qui s'ouvre, faites défiler un peu pour voir cette option.

  4. 4

    Touchez « Ajouter »

    Si vous le souhaitez — renommez en « כספ'לה » (c'est le nom par défaut). Touchez Ajouter.

  5. 5

    ✓ C'est prêt !

    L'icône de כספ'לה apparaît sur l'écran d'accueil. Touchez-la → l'application s'ouvre en plein écran, sans la barre d'adresse du navigateur.

⚠ Remarque : si vous installez via Chrome sur l'iPhone, l'icône ajoute un lien vers le navigateur et non une vraie PWA. Safari = vraie PWA.

Android — via Chrome

  1. 1

    Ouvrez Chrome

    Rendez-vous à l'adresse : kesefle.vercel.app

  2. 2

    Attendez la bannière d'installation

    En général, Chrome affiche une bannière en bas : « Install Kesefle / Installer ». Touchez-la et passez directement à l'étape 4.

    Si la bannière n'apparaît pas, passez à l'étape 3.

  3. 3

    Menu (3 points verticaux) → « Ajouter à l'écran d'accueil » / « Installer l'application »

    Dans le coin supérieur droit de Chrome.

  4. 4

    Touchez « Installer »

    Android installe l'application — elle apparaît dans la liste des applications ainsi que sur l'écran d'accueil.

  5. 5

    ✓ C'est prêt !

    כספ'לה fonctionne comme une vraie application. Elle marche aussi hors ligne (elle affiche le tableau de bord depuis le cache).

💡 Astuce : Samsung Internet et Firefox Android prennent aussi en charge les PWA. La meilleure expérience reste sur Chrome.

Mac — via Safari ou Chrome

  1. 1

    Ouvrez כספ'לה dans Safari ou Chrome

    kesefle.vercel.app

  2. 2A

    Safari : File → Add to Dock

    macOS Sonoma et versions ultérieures. L'icône apparaît dans le Dock — un clic l'ouvre comme une fenêtre d'application.

  3. 2B

    Chrome : dans la barre d'adresse ← icône d'installation

    Sur le côté droit de la barre d'adresse. Elle ressemble à ⊕. Cliquez → « Install ».

  4. 3

    ✓ Application dans le Dock

    כספ'לה s'ouvre dans une fenêtre distincte, avec son propre menu. Vous pouvez la déplacer vers les Spaces, l'ajouter à Stage Manager, etc.

Windows — via Chrome ou Edge

  1. 1

    Ouvrez כספ'לה dans Chrome ou Edge

    kesefle.vercel.app

  2. 2

    Cliquez sur l'icône d'installation dans la barre d'adresse

    À gauche de l'étoile ☆. Elle ressemble à ⊕. Vous ne la voyez pas ? Menu (3 points) → « Installer Kesefle ».

  3. 3

    Cliquez sur « Installer »

    Windows ajoute כספ'לה au menu Démarrer ainsi qu'à la barre des tâches (vous pouvez l'épingler si vous le souhaitez).

  4. 4

    ✓ Une vraie application

    Elle apparaît dans Alt+Tab, dans le Gestionnaire des tâches et dans Paramètres → Applications. Vous pouvez la désinstaller comme n'importe quelle application Windows.

Pourquoi une PWA plutôt qu'une application dans un magasin ?

  • →
    Pas d'approbation Apple/Google — nous n'avons pas besoin d'attendre des semaines l'approbation d'une mise à jour.
  • →
    Mises à jour automatiques — pas besoin de télécharger une nouvelle version. Utilisez la PWA → vous obtenez la toute dernière version.
  • →
    Moins d'espace de stockage — l'application occupe environ 1 Mo au lieu de 50 à 100 Mo.
  • →
    Fonctionne hors ligne — le tableau de bord est enregistré dans le cache local et s'ouvre même sans connexion.
  • →
    Aucune commission à payer — Apple prélève 30 %. Avec une PWA, 100 % du paiement de Pro/Family nous revient et ne va pas à Apple.

Dépannage

Je ne vois pas l'option « Sur l'écran d'accueil » / « Installer l'application »

Vérifiez :
• Vous utilisez le bon navigateur (Safari sur iPhone, Chrome/Edge sur ordinateur, Chrome sur Android)
• Vous naviguez en HTTPS (kesefle.vercel.app, pas http://)
• Votre navigateur est à jour avec la dernière version
• Vous avez visité plusieurs pages du site (certains navigateurs l'exigent avant de proposer l'installation)

L'application affiche une ancienne version même après une mise à jour

Fermez complètement l'application (pas seulement la réduire) et rouvrez-la. Si cela ne suffit pas — supprimez l'application et réinstallez-la via le navigateur. Le Service Worker mettra le cache à jour avec la nouvelle version.

Je ne vois pas le tableau de bord lorsque je suis hors ligne

Vérifiez que vous avez utilisé l'application au moins une fois en étant en ligne — c'est alors que le Service Worker enregistre les pages. Si vous ouvrez l'application pour la première fois hors ligne, elle ne pourra rien afficher (il n'y a rien à charger).

Je veux désinstaller l'application

iPhone : appui long sur l'icône → « Supprimer l'app ». Android : appui long → « Désinstaller » ou via Paramètres → Applications. Mac : faites glisser depuis le Dock → Corbeille. Windows : Paramètres → Applications → Kesefle → Désinstaller.
