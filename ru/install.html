Добавьте כספ'לה
на главный экран телефона

PWA = Progressive Web App — настоящее приложение, работает офлайн, но без необходимости одобрения от Apple/Google и без ожидания 24 часов на загрузку.

iPhone / iPad — через Safari

  1. 1

    Откройте Safari

    Важно: обязательно Safari, а не Chrome на iPhone. iOS не позволяет устанавливать через другие браузеры.

    Перейдите по адресу: kesefle.vercel.app

  2. 2

    Нажмите кнопку «Поделиться»

    Внизу экрана — иконка квадрата со стрелкой вверх ↑

  3. 3

    Прокрутите и выберите «На экран "Домой"»

    (Add to Home Screen) — в открывшемся меню немного прокрутите, и вы увидите эту опцию.

  4. 4

    Нажмите «Добавить»

    При желании измените имя на «כספ'לה» (это значение по умолчанию). Нажмите «Добавить».

  5. 5

    ✓ Готово!

    Иконка כספ'לה появится на главном экране. Нажмите на неё → приложение откроется в полноэкранном режиме, без адресной строки браузера.

⚠ Примечание: если устанавливать через Chrome на iPhone, иконка добавит ссылку на браузер, а не настоящее PWA. Safari = настоящее PWA.

Android — через Chrome

  1. 1

    Откройте Chrome

    Перейдите по адресу: kesefle.vercel.app

  2. 2

    Дождитесь баннера установки

    Обычно Chrome показывает баннер внизу: «Install Kesefle / Установить». Нажмите на него и переходите сразу к шагу 4.

    Если баннер не появился, переходите к шагу 3.

  3. 3

    Меню (три вертикальные точки) → «Добавить на главный экран» / «Установить приложение»

    В правом верхнем углу Chrome.

  4. 4

    Нажмите «Установить»

    Android установит приложение — оно появится в списке приложений и на главном экране.

  5. 5

    ✓ Готово!

    כספ'לה работает как настоящее приложение. Работает и офлайн (показывает дашборд из кэша).

💡 Совет: Samsung Internet и Firefox для Android тоже поддерживают PWA. Лучший опыт — в Chrome.

Mac — через Safari или Chrome

  1. 1

    Откройте כספ'לה в Safari или Chrome

    kesefle.vercel.app

  2. 2A

    Safari: File → Add to Dock

    macOS Sonoma и выше. Иконка появится в Dock — клик открывает её как окно приложения.

  3. 2B

    Chrome: в адресной строке → иконка установки

    В правой части адресной строки. Выглядит как ⊕. Нажмите → «Install».

  4. 3

    ✓ Приложение в Dock

    כספ'לה открывается в отдельном окне со своим меню. Можно перемещать между Spaces, добавлять в Stage Manager и т. д.

Windows — через Chrome или Edge

  1. 1

    Откройте כספ'לה в Chrome или Edge

    kesefle.vercel.app

  2. 2

    Нажмите иконку установки в адресной строке

    Слева от ☆ (звезды). Выглядит как ⊕. Не видите её? Меню (три точки) → «Установить Kesefle».

  3. 3

    Нажмите «Установить»

    Windows добавит כספ'לה в меню «Пуск» и на панель задач (при желании можно закрепить).

  4. 4

    ✓ Настоящее приложение

    Появляется в Alt+Tab, в диспетчере задач и в «Параметры → Приложения». Удалить можно как любое приложение Windows.

Почему PWA, а не приложение из магазина?

  • →
    Никакого одобрения Apple/Google — нам не нужно неделями ждать одобрения обновления.
  • →
    Автоматические обновления — не нужно скачивать новую версию. Откроете PWA → получите самую свежую версию.
  • →
    Меньше места в памяти — приложение занимает ~1 МБ вместо 50–100 МБ.
  • →
    Работает офлайн — дашборд хранится в локальном кэше, открывается даже без подключения.
  • →
    Без комиссий — Apple берёт 30%. С PWA 100% оплаты за Pro/Family идёт нам, а не Apple.

Устранение неполадок

Не вижу опции «Добавить на главный экран» / «Установить приложение»

Проверьте:
• Вы используете правильный браузер (Safari на iPhone, Chrome/Edge на компьютере, Chrome на Android)
• Вы заходите через HTTPS (kesefle.vercel.app, не http://)
• Браузер обновлён до последней версии
• Вы посетили несколько страниц сайта (некоторые браузеры просят это перед предложением установки)

Приложение показывает старую версию даже после обновления

Полностью закройте приложение (не просто сверните) и откройте снова. Если не помогает — удалите приложение и установите заново через браузер. Service Worker обновит кэш до новой версии.

Я не вижу дашборд, когда я офлайн

Убедитесь, что вы хотя бы раз использовали приложение, будучи онлайн — тогда Service Worker сохраняет страницы. Если открыть приложение впервые офлайн, оно не сможет ничего показать (нечего загружать).

Я хочу удалить приложение

iPhone: долгое нажатие на иконку → «Удалить приложение». Android: долгое нажатие → «Удалить» или через настройки → Приложения. Mac: перетащите из Dock → Корзина. Windows: «Параметры» → «Приложения» → Kesefle → «Удалить».

Ещё не создали аккаунт?

Регистрация через Google → личная таблица в Drive → начните вести учёт.
